 
## Project first RUN

 1. - `git clone git@github.com:dusant94/Ambulance.git` 

 2. - `cd apartment-api/`

 3. - `composer install`

 4. - `cp .env.example .env`

 5. - `php artisan key:generate`

 6. - `php artisan migrate`

 7. - `php artisan serve`

 9. - `npm install & npm run dev`


### Commands for seeding fake database data

 - php artisan db:seed
